Le fichier Zip regroupe l'ensemble des signaux collectés durant le projet. Il est présenté sous trois dossiers:
- Data_arduino : contient l'ensemble des signaux capturés directement sur l'arduino.
- Tiny_motion_train_data : regroupe l'ensemble des signaux capturés avec TinyMotion Trainer.

caractéristiques des signaux:

Fréquence d'échantillonnage : 114Hz (fréquence par défaut de la carte)
Nombre d'échantillons par signal : 50 échantillon
Threshold : Seuil utilisé pour déterminer si présence d'un mouvement
